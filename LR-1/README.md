### Название проекта

**Text Analyzer (Консольная программа для анализа текста на Python)**

### Назначение

Программа предназначена для анализа введённого пользователем текста и вычисления основных статистических характеристик.

### Выбранная методология

**Инкрементальная (итерационная) разработка** с элементами **Agile-подхода**.

### Этапы работы

1. Анализ требований
2. Проектирование структуры программы
3. Реализация базового функционала
4. Добавление дополнительных возможностей
5. Тестирование и отладка

### Применённые методы

* анализ требований;
* декомпозиция задачи;
* алгоритмизация;
* модульное проектирование;
* обработка строк;
* использование словарей Python;
* сортировка данных;
* тестирование и отладка.

### Возникшие сложности

1. **Корректный подсчёт слов** при наличии знаков препинания.\
   **Решение:** реализована очистка текста от знаков препинания перед анализом.
2. **Подсчёт предложений** при различных окончаниях.\
   **Решение:** использован анализ символов `.`, `!`, `?`.
3. **Поиск самого длинного слова** при наличии спецсимволов.\
   **Решение:** предварительная нормализация текста.

### Способы запуска

1. Сохранить файл как `text_analyzer.py`
2. Запустить в консоли:

python text_analyzer.py
